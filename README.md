# Triple R Recycling Limited

Triple R Recycling Limited is a social enterprise based in Abuja, Nigeria, dedicated to transforming plastic waste into valuable resources. Our mission is to build a cleaner, greener future by incentivizing recycling, empowering communities, and promoting sustainable practices.

## 🌍 Mission

- **Reduce** plastic pollution through convenient and rewarding recycling services.
- **Reuse** materials by repurposing plastics into new products.
- **Recycle** efficiently using innovative technology and community engagement.

## 🚀 Features

- **End-to-End Recycling Workflow:** From collection and sorting to baling and transportation.
- **Incentive-Based Model:** Cash or value rewards for every kilogram of plastic recycled.
- **Community Engagement:** School programs, awareness campaigns, and local partnerships.
- **Smart Tech Tracking:** Digital tools for transparent waste collection and reward distribution.
- **Women & Youth Empowerment:** 70% of workforce are women; focus on job creation in underserved areas.
- **Educational Outreach:** Waste literacy programs for schools and communities.

## 🖥️ Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **Routing:** React Router
- **State Management:** React Hooks
- **Forms:** Custom forms with [FormSubmit](https://formsubmit.co/)
- **Icons:** Lucide React
- **Animations:** Framer Motion
- **Other:** Swiper.js for carousels, React Helmet for SEO

## 🛠️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16+ recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository:**

   ```sh
   git clone https://github.com/yourusername/tripleR.git
   cd tripleR
   ```

2. **Install dependencies:**

   ```sh
   npm install
   # or
   yarn install
   ```

3. **Start the development server:**

   ```sh
   npm run dev
   # or
   yarn dev
   ```

4. **Open in your browser:**
   Visit [http://localhost:5173](http://localhost:5173) (or the port shown in your terminal).

### Building for Production

```sh
npm run build
# or
yarn build
```

### Linting

```sh
npm run lint
# or
yarn lint
```
